title: Les travaux publics de la ville de Périgueux au xive siècle



== Chapitre premier étude générale ==

1.
Organisation. – Département des travaux publics. – Commande des travaux : évaluation, devis. – Exécution : le consul délégué ; le maître de l’œuvre, ses attributions ; le directeur de la « manobra ». – Comptabilité. – Les travaux publics pendant une saisie royale.

2.
Ressources. – Existence d’un budget. – Budget ordinaire. – Ressources extraordinaires : 1° impôt spécial, son vote ; impôt direct ; impôt indirect : la vinade, son importance ; prestations.
Perception ; résistances des laïcs, des gens d’église ; attitude de la municipalité. – 2° Dons du roi.

3.
Droits. – La réquisition.
Qui a le droit de réquisitionner ?
Formalités ; tout le monde peut être réquisitionné ; on réquisitionne les immeubles, les meubles et les personnes.
Indemnités des réquisitions, formalités.
L’indemnité.
Cas de force majeure ; paiement par exécution de la réparation ; zone inindemnisable.

4.
Rapports de la ville avec les ouvriers. – Les métiers des travaux.
Les ouvriers.
Inégalité des métiers.
Les manœuvres.
Les femmes ; quelques-uns de leurs travaux ; on leur donne même des travaux à prix fait.
Les enfants.
L’armurier.
La « manœuvre ».
Les pourboires. – Conditions du travail : extrême division du travail ; travail à la journée, limitation des salaires par le consulat ; travail à prix fait, les épingles ; procédé mixte. – Bienveillance de la ville envers les ouvriers.
Jours fériés. – Droits protecteurs perçus sur les ouvriers étrangers.
Les accidents du travail ; accidents causés aux tiers.

5.
Les matériaux. – La pierre : carrières, extraction, taille, qualité ; le sable ; le gravier ; la terre ; la chaux ; le four des Prêcheurs ; le four de la ville ; chauffage du four ; le mortier ; le fer ; le bois. – Outillage : les outils de la pierre, du sable et du mortier ; la cigogne ; la « vernha » ; outils du fer : les échelles ; location d’outils. – Transport des matériaux.
Emmagasinage.

Propriété des poids.

Nettoyage des rues ; mesures de propreté.

Les fontaines.

8.
Les fortifications. – Le problème topographique.
Le mur d’enceinte : construction, escaliers.
Les hourds ; procédé mixte : les piles.
Défense du mur d’enceinte.
Les tours : situation, différents types, construction ; guérites, mâchicoulis.
Accès du mur d’enceinte.
Les portes .⁎ principales portes, différents types, fermeture.
Défense des portes : les barbacanes, grandes dimensions de quelques-uns de ces ouvrages ; les hourds.
Défenses accessoires du mur d’enceinte : les fossés, zone militaire, les petits murs, les avant-murs.
Barrières, chevaux de frise, tranchées.

La défense du côté de l’eau : la tour de Boucherie, le moulin Saint-Front, l’abreuvoir, le pont de Tournepiche, le port de Graule, la tour de Gordonnet.
L’opinion des contemporains sur cette défense.

Les ponts-levis : le tablier, les chaînes, les verges, la cigogne, les clairières.
